# Security Policy

## Supported versions

Only the latest published npm version is considered fully supported.

## Reporting a vulnerability

Please do **not** open a public issue for security vulnerabilities.

Contact:

- GitHub Security Advisories (preferred)
- or open a private report to maintainers

Include:

- affected command/workflow/prompt
- reproduction steps
- expected vs actual behavior
- potential impact

## Response targets

- initial acknowledgment: within 3 business days
- triage and severity: within 7 business days
- fix plan: as soon as triage is complete
