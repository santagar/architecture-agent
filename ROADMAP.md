# Roadmap

## Near-term

- Improve determinism guidance for scoring/review prompts (reduced score drift).
- Add richer CLI smoke tests for all workflow launch modes.
- Add optional JSON output mode for `launch` to ease IDE/task integrations.

## Mid-term

- Add migration notes tooling for prompt/workflow breaking changes.
- Add command to validate target repository prerequisites before launch.
- Add optional constraints profiles (strict governance, fast discovery, modernization).

## Long-term

- Multi-repository orchestration helpers for solution-landscape runs.
- Diff-aware report generation quality checks.
- Wider template catalog for operational docs and architecture decision records.
